## 주요 기능 구현 내용

### 주문 생성 로직
1. 상품 조회 → 가격/도매처 정보 스냅샷 저장 (나중에 상품 수정돼도 주문 기록 보존)
2. 주문번호 자동 생성: `ORD-YYYYMMDD-XXXX`
3. 도매처 `notify_type = 'email'` → 주문 즉시 Resend 이메일 자동 발송 + 상태 `forwarded`
4. `notify_type = 'manual'` → 상태 `received` → 관리자가 수동 발송

### 인증 흐름
- `middleware.ts`: `/admin/*` 접근 시 Supabase 세션 확인 → 없으면 `/admin/login` 리다이렉트
- Service Role 키는 서버 전용 API에서만 사용 (클라이언트 노출 없음)

### 키오스크 최적화
- **Wake Lock API**: 화면 자동 꺼짐 방지
- **3분 자동 리셋**: 터치/클릭 없으면 메인 화면으로 자동 새로고침
- **터치 최적화**: 최소 탭 영역 44px, 확대 방지 `viewport` 설정
